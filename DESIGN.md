# DESIGN.md

## 气质与意象
学术殿堂的现代演绎——清晨图书馆透过的暖黄灯光洒在深色胡桃木书桌上，空气中有纸张与墨水的气味，周围是整齐排列的书架与安静的学子。信息密集但井然有序，工具感强但不冰冷。

## 视觉策略
- 图形语言：克制的几何线条 + 微妙的渐变，呼应"知识的秩序感"
- 卡片采用细微阴影 + 1px 边框，悬浮时提升阴影层级
- 数据图表使用平滑曲线 + 半透明面积填充

## 配色方案
- 主色：#1e293b（深海军蓝/slate-800）——学术沉稳、可信赖
- 强调色：#f59e0b（琥珀金/amber-500）——代表"金榜题名"的期许与成就感
- 辅助色：#3b82f6（蓝色/blue-500）——信息链接与交互引导
- 背景：#f8fafc（雾白/slate-50）——舒适阅读
- 卡片：#ffffff ——干净承载信息
- 专硕标识色：#10b981（翡翠绿/emerald-500）
- 学硕标识色：#8b5cf6（紫罗兰/violet-500）

## 字体排版
- 标题：思源黑体/Noto Sans SC（Google Fonts），600/700 权重
- 正文：系统默认 -apple-system, sans-serif，400 权重
- 数据数字：等宽特性字体用于分数线和录取人数展示
- 排版节奏：标题 2rem/1.5rem/1.125rem 三级，正文 0.875rem，行高 1.6

## 动效与交互
- 卡片悬浮：translateY(-2px) + shadow 扩展，duration 200ms ease-out
- 页面切换：fade-in 300ms
- 图表加载：从左到右描绘动画 800ms
- 搜索结果：交错淡入 stagger 50ms
- Tab 切换：下划线滑动过渡 200ms

## 页面结构
- 顶部固定导航栏：Logo + 功能入口 + 搜索框
- 首页 Hero：大搜索框 + 功能卡片矩阵（院校查询、AI择校、专业查询、分数线查询、院校对比、考研资讯）
- 院校查询页：左侧筛选面板（省份/类型/层次）+ 右侧院校卡片列表
- AI择校页：左侧信息表单 + 右侧流式对话输出
- 专业查询页：学硕/专硕 Tab 切换 + 门类筛选 + 专业列表
- 院校详情页：基本信息 + 开设专业 + 历年录取数据图表
- 院校对比页：最多3所院校并排对比

## 设计禁忌
- 不要使用科技蓝紫渐变作为主色
- 不要使用大圆角（>12px）卡片
- 不要在数据密集区域使用全彩背景
- 不要使用卡通风格插图
- 禁止信息层级混乱——数据必须可扫描
